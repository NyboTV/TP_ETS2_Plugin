# V5.1.9 - Stability Update & Mac/Linux Fixes

This update brings important under-the-hood improvements and resolves critical issues for macOS and Linux users.

## 🛠️ Bug Fixes & Improvements
- **Linux & Mac Support Restored**: Fixed a critical issue that caused the plugin to crash immediately on Linux and Mac systems. The plugin now runs smoothly again across all operating systems.
- **Mac Installation Fix**: Resolved an issue where the Mac version was missing from recent download files. The `.tpp` file for Mac is now properly included and available again.
- **Enhanced Error Diagnostics (Logging)**: We have significantly expanded the plugin's diagnostic system. If you experience issues during setup or while using Touch Portal in the future, the plugin now writes much more detailed logs. This will help us assist you faster and more accurately.
- **Optimized File Structure**: Internal cleanup ensures that the plugin is now packaged even cleaner and more efficiently.
